## Project Description
This project analyzes the "Students Performance in Exams" dataset from Kaggle.
It covers descriptive statistics, data cleaning, exploratory data analysis (EDA),
correlation analysis, and professional data visualizations — all implemented in Python.
 
## Dataset
- **Source:** [Kaggle – Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)
- **File:** `StudentsPerformance.csv`
- **Records:** 1000 students
- **Features:** gender, race/ethnicity, parental education, lunch, test prep, math/reading/writing scores

1. Install dependencies: `pip install pandas numpy matplotlib seaborn scipy`
